- Splash Screen (3s)
- Tabs: Previsão e Mapa

- Google Maps
- QR Code para trocar cidade

## Tecnologias

- Java
- Material Design
- ViewPager2 + TabLayout
- Volley + ZXing



## Como Executar

1. Clone: `git clone https://github.com/renanpires/previsao-do-tempo-unipar.git`
2. Abra no Android Studio
3. Sync Gradle
4. Execute
